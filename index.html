What InventoryOS does

InventoryOS helps resellers manage stock, sales and dispatch in one web app.

InventoryOS is a browser-based inventory management application for UK resellers. Users can add products, track stock locations, scan barcodes, record sales, manage expenses, upload receipts, prepare shipping labels and view business reports from one account.

When a user chooses to connect Gmail, InventoryOS only uses Gmail access to find shipping-label emails and label attachments for the Dispatch Centre. This helps users import postage labels into their packing and printing workflow.

Gmail connection is optional. InventoryOS does not use Gmail access for advertising, marketing, selling user data or reading unrelated emails. Users can disconnect Gmail access at any time.

InventoryOS works alongside the marketplaces you already use. It does not replace your listings or marketplace account.

InventoryOS is independent inventory and operations software. You can use it alongside Vinted, eBay, Facebook Marketplace, Amazon returns, car-boot sales and other channels.

Gmail connection is optional. It is used to help import shipping-label emails and label attachments into the Dispatch Centre, so sellers can prepare and print orders faster. InventoryOS does not use Gmail access for advertising, marketing, selling user data or reading unrelated emails.

No. You can type or search normally. A USB, wireless or Bluetooth scanner can make higher volumes quicker because most scanners behave like a keyboard.
